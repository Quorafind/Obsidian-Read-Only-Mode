# Obsidian Read-Only Mode

> [!note]
> This plugin is only work for Obsidian.

## Usage

> [!warning]
> Via action button
> 1. Open the Live Preview/Source mode.
> 2. Toggle the Read-Only Mode.
> 3. Enjoy it.

> [!note]
> Via command
> 1. Open the command palette.
> 2. Type `Read-Only Mode`.
> 3. Enjoy it.
